# 歯車｜創作語彙集

芥川龍之介「歯車」から抽出した創作に使える語彙・表現。

---

## 視覚異常・幻視の描写

| 表現 | 用法 |
|------|------|
| 半透明の歯車 | 視界を侵食する幻視の具体化 |
| 絶えずまわっている | 止められない、制御不能な状態 |
| 視野を塞ぐ | 現実認識の阻害 |
| 次第に数を増し | 症状の悪化、侵食の進行 |
| 妙なものを見つけ出した | 異常の発見、不穏な気づき |

**応用例**: 不安や狂気の進行を視覚的に表現したいとき

---

## 神経・精神状態の比喩

| 表現 | 解説 |
|------|------|
| 僕の神経は鼠のように静かに僕自身を噛んでいる | 自己破壊的な内面の動き。「静かに」が不気味さを増す |
| 薄氷を踏んでいるような | 危うい精神状態、いつ壊れてもおかしくない |
| 何か僕の背後にあるものの存在を感じた | 漠然とした脅威、追われる感覚 |
| 絶えず脅かされつづけた | 持続する不安、安息のなさ |
| 心の底に潜んでいる | 無意識の領域、抑圧されたもの |

---

## 不安・恐怖の表現

| 表現 | 用法 |
|------|------|
| ぼんやりした不安 | 対象のない、漠然とした恐れ |
| 妙に超自然的な | 説明のつかない異様さ |
| 何とも言われない苦痛 | 言語化できない苦しみ |
| じっと僕を見つめている | 監視されている感覚、逃れられなさ |
| 部屋の隅に誰かが | 不在の存在、幻覚と現実の境界 |
| 追われているような気がした | 被害妄想的な感覚 |

---

## 偶然・符号の表現

| 表現 | 解説 |
|------|------|
| 偶然の一致 | 無関係な事象が意味を持って迫る |
| 不吉な前兆 | 予感、予告 |
| 何か暗示しているような | 隠された意味への恐怖 |
| ふと〜を思い出した | 無意識の連想、制御できない想起 |
| 〜という言葉が頭を離れない | 強迫的な反復 |
| 至るところに〜を見出した | パターン認識の暴走 |

---

## 色彩表現

| 色 | 表現・用法 |
|------|------|
| 黄色 | 狂気、病、不吉。「黄色いタクシー」が繰り返し現れる |
| 赤 | 血、炎、破滅。「赤い光」は終末的イメージ |
| 半透明 | 現実と幻覚の境界、曖昧さ |
| 薄暗い | 不穏、不安、見えないもの |

## 身体感覚

| 表現 | 用法 |
|------|------|
| 頭が重い | 精神的疲弊の身体化 |
| 眼を休ませようとした | 無駄な抵抗、制御の試み |
| 寒気を感じた | 恐怖・不安の身体反応 |
| 足もとの覚束ない | 不安定さ、崩壊の予感 |
| 息苦しさを感じた | 圧迫感、閉塞感 |
| 動悸が激しくなった | 緊張、恐怖の高まり |

## 時間・空間の歪み

| 表現 | 用法 |
|------|------|
| いつの間にか | 時間感覚の喪失 |
| 何時間経ったかわからない | 時間の歪み |
| どこにいても | 逃げ場のなさ、遍在する脅威 |
| 夢の中のような | 現実感の喪失 |
| 境界が曖昧になる | 狂気への接近 |

## 文体・リズム

**短文の連打**
> 僕は立ち上がった。窓を開けた。外は暗かった。

緊張感、息苦しさを生む。

**疑問形の挿入**
> 妙なものを？——と云うのは

読者を引き込む。語り手の動揺を示す。

**ダッシュ（——）の多用**
> それは——いや、何でもない。

思考の断絶、言い淀み、言語化の困難を表す。

**反復**
> 黄色いタクシー。また黄色いタクシー。

強迫的な認知、逃れられなさ。

## 動詞・述語の選択

| 弱い表現 | 芥川的表現 | 効果 |
|----------|------------|------|
| 怖かった | 脅かされつづけた | 持続性、受動性 |
| 見た | 見つけ出した | 発見のニュアンス |
| 思った | 頭を離れなかった | 制御不能性 |
| 不安だった | 薄氷を踏んでいた | 具体化、身体化 |
| 苦しかった | 静かに噛んでいた | 比喩による異化 |

## 創作への応用パターン

### 1. 日常の不穏化

普通の事物（タクシー、本のタイトル、数字）に意味を持たせる。

```
また同じ数字だ。偶然だ。偶然のはずだ。
```

### 2. 視覚の侵食

視界に異物が入り込む描写。

```
視野の端に何かがちらついた。振り向いても何もない。
だが確かに、そこに——
```

### 3. 自己言及的不安

自分が今していることへの違和感。

```
僕はこれを書いている。書いているということは、まだ正気だということだ。
そう信じたい。
```

### 4. 偶然の必然化

無関係な出来事を結びつける語り。

```
朝、鴉が鳴いた。午後、彼女から電話があった。
夜、僕は——いや、これは偶然だ。偶然に決まっている。
```

### 5. 未完の文

言い切らないことで不安を残す。

```
あの時、僕が見たものは——
```

## 参考：原文からの抽出

> 僕の視野のうちに妙なものを見つけ出した。妙なものを？——と云うのは絶えずまわっている半透明の歯車だった。

> 僕はかう云ふ偶然の一致に絶えず脅かされつづけた。

> 僕はペンを持ったまま、いつも薄暗い部屋の隅に誰かがじっと僕を見つめているような気がした。

> 僕の神経は鼠のように静かに僕自身を噛んでいるのに相違なかった。

> それは僕の一生のうちに一度も経験したことのない、妙に超自然的な、そして又妙に美しい光だった。

> 誰か僕の眠っているうちにそっと絞め殺してくれるものはないか。
