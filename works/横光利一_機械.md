# 機械｜創作語彙集

横光利一「機械」から抽出した創作に使える語彙・表現・技法。

---

## 意識の流れ（長文構築）

句読点を減らし、接続詞で思考を繋げていく技法。

### 基本パターン

```
[事実] + のだが + [留保] + とはいえ + [別の考え] + かもしれぬし + [結論の先送り]
```

**原文の例**:

> しかし、私にしてみればただこの仕事を覚え込んでおくだけでそれで生涯の活計を立てようなどとは謀んでいるのでは決してないのだが、そんなことをいったって軽部には分るものでもなし、また私がこの仕事を覚え込んでしまったならあるいはひょっこりそれで生計を立てていかぬとも限らぬし、いずれにしても軽部なんかが何を思おうとただ彼をいらいらさせてみるのも彼に人間修養をさせてやるだけだとぐらいに思っておればそれで良ろしい

**応用例**:

```
【通常】
私は彼女を愛していなかった。しかし、嫌いでもなかった。

【機械的】
私は彼女を愛していたのかいなかったのかそんなことは今となっては分るものでもないのだが、かといって嫌いだったのかというとそれもまた違うような気がするし、いずれにしても愛だの嫌悪だのといったことを今さら持ち出してみたところで何の役にも立つものではないのである。
```

### 接続表現カタログ

| 表現 | 機能 |
|------|------|
| のだが | 逆接・留保 |
| とはいえ | 譲歩 |
| かもしれぬし | 可能性の付加 |
| いずれにしても | まとめ |
| そうかといって | 別の観点 |
| それにしても | 話題の転換 |
| しかしながら | 強い逆接 |
| 〜ものでもなし | 否定的付加 |

## 認識の不確かさ

夢と現実、記憶と想像の境界を曖昧にする表現。

### パターン

| 表現 | 用法 |
|------|------|
| 〜であったのか〜であったのか | 二者択一の不確定 |
| 全く分らなくなって来た | 認識の崩壊 |
| 〜かもしれない | 可能性の提示 |
| 〜ではないかと思われて | 推量の重ね |
| 〜にちがいないのだ | 確信のない断定 |

**原文の例**:

> しかし、困ったことには考えているうちにそれは私の夢であったのか現実であったのか全く分らなくなって来たことだ。

> もしかすると屋敷を殺害したのは私かもしれぬのだ。

**応用例**:

```
彼女が去ったのは昨日だったか一昨日だったか、考えているうちに全く分らなくなって来て、ひょっとするとまだ去ってはいないのかもしれぬとさえ思われて来るのだ。
```

## 疑惑・監視の語彙

互いを疑い、監視し合う関係を描く表現。

### 動詞

| 動詞 | ニュアンス |
|------|------------|
| 疑う | 基本形 |
| 疑り出す | 疑惑の開始 |
| 見詰める | 監視の視線 |
| 睨みつける | 敵意ある監視 |
| 眼を光らせる | 警戒 |
| 注意をそそぐ | 観察 |
| 気附く | 発見 |
| 見破る | 看破 |

### 表現パターン

**原文の例**:

> 彼は私が棚の毒薬を手に取って眺めているともう眼を光らせて私を見詰めている。私が暗室の前をうろついているともうかたかたと音を立てて自分がここから見ているぞと知らせてくれる。

> いったい本当はどちらがどんな風に私を思っているのかますます私には分らなくなり出した。しかし事実がそんなに不明瞭な中で屋敷も軽部も二人ながらそれぞれ私を疑っているということだけは明瞭なのだ。

**応用例**:

```
彼が私の机の前を通るたびに眼を光らせて書類を見ているのを私は気附いていたし、私が彼のパソコンの画面を覗き込むたびに彼が身構えるのも分っていた。互いが互いを疑っているということだけは確かなのだ。
```

## 劇薬・化学の語彙

工場の危険な環境を描く専門用語。創作では比喩的にも使える。

### 化学物質

| 物質 | 作用 |
|------|------|
| 塩化鉄 | 金属を腐蝕させる |
| 臭素 | 咽喉を破壊する |
| 重クロム酸アンモニア | 致死性の劇薬 |
| カセイソーダ | 洗浄に使う |
| アニリン | 着色に使う |
| ビスムチル（蒼鉛） | 化合物の材料 |
| カルシウム | 粉末として登場 |

### 身体への作用

**原文の例**:

> この穴へ落ち込むと金属を腐蝕させる塩化鉄で衣類や皮膚がだんだん役に立たなくなり、臭素の刺戟で咽喉を破壊し夜の睡眠がとれなくなるばかりではなく頭脳の組織が変化して来て視力さえも薄れて来る。

**比喩的応用**:

```
彼女の言葉は塩化鉄のように私の自尊心を腐蝕させていった。
その職場は臭素のように精神を蝕み、やがて思考する力さえ奪っていくのだ。
```

## 暴力の描写

身体の動きを外側から冷静に描写する技法。

### 動詞カタログ

| 動詞 | 動作 |
|------|------|
| 捻じ伏せる | 押さえつける |
| 引き摺り出す | 引っ張り出す |
| 投げつける | 投げる |
| 殴りつける | 殴る |
| 蹴りつける | 蹴る |
| 押し倒す | 倒す |
| 馬乗りになる | 上から押さえる |
| 締めつける | 絞める |
| ぶちあてる | ぶつける |

### 被害者視点の描写

**原文の例**:

> 軽部の膝骨が屋敷の背中を突き伏せる度毎にまた直ぐべたべたと崩れてしまって着物の捲れあがった太った赤裸の両足を不恰好に床の上で藻掻かせているだけなのだ。

身体の部位（膝骨、背中、両足）と動き（突き伏せる、崩れる、藻掻く）を物理的に描写。感情は書かない。

**応用例**:

```
【感情的】
彼は痛みに耐えながら必死に抵抗した。

【機械的】
彼の肩が床に押しつけられる度に腕が不自然な角度に捻じ曲がり、口からは意味のない音が漏れるだけだった。
```

## 自己認識の崩壊

「私」が「私」を把握できなくなる表現。

### パターン

| 表現 | 状態 |
|------|------|
| 私が分らなくなって来た | 自己認識の喪失 |
| 自分を持てあます | 自己制御の不能 |
| 私に代って私を審いてくれ | 他者への委譲 |
| 私の知っていよう筈がない | 自己知の否定 |

**原文の例**:

> 私はもう私が分らなくなって来た。私はただ近づいて来る機械の鋭い先尖がじりじり私を狙っているのを感じるだけだ。誰かもう私に代って私を審いてくれ。私が何をして来たかそんなことを私に聞いたって私の知っていよう筈がないのだから。

**応用例**:

```
私は自分が何を望んでいるのか分らなくなって来た。私はただ締め切りという機械がじりじり私を追い詰めてくるのを感じるだけだ。誰かもう私に代って私の人生を決めてくれ。私が何をしたいのかそんなことを私に聞いたって私の知っていよう筈がないのだから。
```

## 機械のメタファー

人間関係や運命を「機械」として描く表現。

### 原文の例

> そうして私達は互に疑い合いながらも翌日になれば全部の仕事が出来上って楽々となることを予想し、その仕上げた賃金を貰うことの楽しみのためにもう疲労も争いも忘れてその日の仕事を終えてしまうと、いよいよ翌日となってまた誰もが全く予想しなかった新しい出来事に逢わねばならなかった。

予想と結果のずれ。機械的に進行する因果関係。

> 私はただ近づいて来る機械の鋭い先尖がじりじり私を狙っているのを感じるだけだ。

運命・宿命としての機械。

### 応用パターン

```
私たちは互いを愛していると信じながらも毎週末になれば喧嘩をし、その喧嘩の後の仲直りの甘さのために疲労も怒りも忘れて眠ってしまうと、いよいよ翌週になってまた同じ喧嘩を繰り返さねばならなかった。それはまるで機械のように正確に作動する不幸であった。
```

## 善良さの描写

主人の「善良すぎる」性格を描く表現。

### 原文の例

> 主人の肉体から出て来る光りに射抜かれてしまったわけだ。奇蹟などというものは向うが奇蹟を行うのではなく自身の醜さが奇蹟を行うのにちがいない。

善良な人の前で自分の醜さを感じる。

> 困っているものには自分の家の地金を買う金銭まで遣ってしまって忘れている。こういうのをこそ昔は仙人といったのであろう。しかし、仙人と一緒にいるものは絶えずはらはらして生きていかねばならぬのだ。

善良すぎることの困難さ。

## 秘密と暴露の構造

秘密を守る/盗む関係を描く表現。

### 原文の例

> すべて秘密というものはその部分に働く者の慢心から洩れるのだと気がついたのはそのときの何よりの私の収穫であったであろう。

秘密が漏れるメカニズム。

> 向うが秘密を盗もうとするならこちらはそれを隠したってかまわぬであろう。

秘密をめぐる攻防。

## 創作への応用パターン

### 1. 長文で内面を描く

```
【通常】
彼は迷っていた。転職すべきか、残るべきか。どちらにも理由があった。

【機械的】
彼は転職すべきなのかそれとも残るべきなのかそんなことは誰に聞いたところで分るものでもないのだが、かといって自分で決められるのかというとそれもまた怪しいもので、転職すれば後悔するかもしれぬし残れば残ったで後悔するかもしれぬし、いずれにしてもどちらを選んでも後悔するのならば早く選んでしまった方が良いのだがそれがなかなか出来ないのである。
```

### 2. 疑惑の循環を描く

```
AはBを疑い、BはCを疑い、CはAを疑っていた。誰が本当の敵なのか分らないまま、三人は互いを監視し続けた。しかし事実がそんなに不明瞭な中で三人とも疑われているということだけは明瞭なのだ。
```

### 3. 労働の疎外を描く

```
この仕事を続けていると画面の光で視力がだんだん役に立たなくなり、椅子の圧迫で腰を破壊し夜の睡眠がとれなくなるばかりではなく思考の組織が変化して来て創造力さえも薄れて来る。
```

### 4. 結末を曖昧にする

```
私がそれをしたのかしなかったのか、今となっては私にも分らない。私はただ結果だけを見ている。誰かもう私に代って私を審いてくれ。
```

## 芥川「歯車」との対比

| 歯車 | 機械 |
|------|------|
| 「僕」という一人称 | 「私」という一人称 |
| 意味の過剰（象徴が迫る） | 認識の崩壊（事実が分からない） |
| 偶然が必然に見える | 因果が機械的に作動する |
| 自意識の肥大 | 自意識の解体 |
| 「誰か僕の眠っているうちに」 | 「誰かもう私に代って私を審いてくれ」 |

両者を学ぶことで、「意味に追い詰められる」と「認識が崩壊する」という二つの内面崩壊パターンが身につく。
