# 頭ならびに腹｜創作語彙集

横光利一「頭ならびに腹」から抽出した創作に使える語彙・表現・技法。

---

## 主語の転倒

通常の主語を逆転させ、受動的・周縁的なものを主語にする。

| 通常の表現 | 横光的表現 | 効果 |
|------------|------------|------|
| 列車が駅を通過した | 小駅は黙殺された | 通過される側の視点 |
| 彼が窓を見た | 窓が彼の視線を受けた | 物の存在感 |
| 乗客が騒いだ | 騒音が車内を満たした | 抽象の具象化 |

**応用例**:
```
彼女が去った。
→ 部屋は見捨てられた。
```

```
雨が降り始めた。
→ 街路は雨に打たれ始めた。
```

---

## 物質化（抽象→具象）

抽象的な感情・概念を、物理的な存在として扱う。

| 抽象 | 物質化表現 |
|------|------------|
| 空腹 | 空腹が膨れ上がった |
| 退屈 | 退屈が車内に沈殿した |
| 緊張 | 緊張が肩に凝った |
| 期待 | 期待が胸に溜まった |
| 不安 | 不安が胃の底に溜まった |

**応用例**:
```
彼は退屈だった。
→ 退屈が彼の背中に貼りついていた。
```

```
彼女は不安だった。
→ 不安が喉元まで込み上げていた。
```

## 断定的直喩

「〜のようだ」ではなく、比喩を断定的に書く。

| 通常の直喩 | 断定的直喩 |
|------------|------------|
| 石のように黙っていた | 石のやうに黙殺された |
| 機械のように動いた | 機械として動いた |
| 弾丸のように走った | 弾丸が走った |

**技法**: 「ようだ」「みたいだ」を省略し、AとBを直接接続する。

**応用例**:
```
彼は狼のように孤独だった。
→ 狼の孤独が、彼の中で遠吠えしていた。
```

---

## 映画的カット

短い文を並列し、カメラのカット割りのように場面を構成する。

**基本パターン**:
```
[全景] → [中景] → [クローズアップ]
```

**原文の例**:
```
真昼である。（状況設定・光）
特別急行列車は満員のまま全速力で馳けていた。（全景・動き）
沿線の小駅は石のやうに黙殺された。（通過される対象）
```

**応用例**:
```
夕暮れだった。
オフィスビルは残業の光を灯していた。
彼の机は書類に埋もれていた。
```

## 並列スケッチ

「或る者は〜、或る者は〜」という並列で、群衆を描く。

**原文の例**:
```
或る者は辨當の蓋を開けた。
或る者は窓から首を出した。
或る者は新聞を擴げた。
```

**応用例**:
```
或る者はスマートフォンを見た。
或る者はイヤホンを耳に押し込んだ。
或る者は目を閉じた。
誰も誰とも話さなかった。
```

## 体言止め

名詞で文を終わらせ、スピード感と断片性を出す。

| 通常 | 体言止め |
|------|----------|
| 真昼だった | 真昼。 |
| 満員だった | 満員。 |
| 全速力で走っていた | 全速力。 |

**連続使用の例**:
```
真昼。満員。全速力。
窓の外、田園。遠くに山。
車内、沈黙。時折、咳。
```

---

## 現在形の効果

過去形ではなく現在形を使い、読者を「今ここ」に引き込む。

| 過去形 | 現在形 |
|--------|--------|
| 列車は走っていた | 列車は走っている |
| 真昼だった | 真昼である |

**応用**: 緊迫した場面、読者を巻き込みたい場面で現在形に切り替える。

## 感覚動詞のカタログ

「頭ならびに腹」で使われる感覚的な動詞。

### 視覚
- 黙殺する（見ないことの能動化）
- 流れる（風景の動き）
- 過ぎる
- 映る

### 聴覚
- 轟く
- 響く
- 沈黙する

### 身体感覚
- 揺れる
- 傾く
- 膨れる
- 凝る

### 運動
- 馳ける（「走る」より速度感）
- 滑る
- 突き進む

## 「頭」と「腹」の対比語彙

| 頭（知性・理性） | 腹（本能・欲望） |
|------------------|------------------|
| 思考する | 欲する |
| 判断する | 求める |
| 分析する | 貪る |
| 推論する | 渇く |
| 計算する | 飢える |
| 認識する | 感じる |

**応用**: 人物を描くとき、「頭」と「腹」の二重性を意識する。

```
彼の頭は冷静に状況を分析していた。
しかし彼の腹は、逃げろと叫んでいた。
```

## 速度の表現

列車のスピードを表す語彙。

| 表現 | ニュアンス |
|------|------------|
| 全速力 | 最大速度 |
| 馳ける | 疾走する（古語的） |
| 突き進む | 直線的な力 |
| 滑る | なめらかな高速 |
| 飛ぶ | 地面から離れるような速さ |

**派生**: 速度を使った時間圧縮

```
駅が過ぎた。また駅が過ぎた。
彼の人生も、そのように過ぎていくのだろう。
```

## 無生物の擬人化パターン

| 無生物 | 擬人化表現 |
|--------|------------|
| 駅 | 黙殺された、見捨てられた、忘れられた |
| 列車 | 吠える、唸る、息をつく |
| 窓 | 見つめる、受け止める、拒む |
| 座席 | 抱える、支える、沈む |
| 時間 | 流れる、滞る、圧縮される |

## 創作への応用パターン

### 1. 冒頭を新感覚派的に書く

```
【従来】
その日は暑い夏の日だった。私は電車に乗って会社に向かっていた。

【新感覚派的】
猛暑。満員電車は汗を運んでいた。乗客たちは互いを黙殺した。
```

### 2. 群衆シーンを断片で描く

```
或る者はスマートフォンを握りしめていた。
或る者は吊革に体重を預けていた。
或る者は目を閉じて、どこか遠くにいた。
電車は、これらすべてを運んでいた。
```

### 3. 内面を外形で描く

```
【従来】
彼は悲しかった。

【新感覚派的】
彼の肩が落ちた。視線が床に吸い込まれた。
呼吸が浅くなった。悲しみが、彼の輪郭を溶かしていた。
```

### 4. 速度で圧縮する

```
会議。昼食。会議。電話。会議。
夕方になっていた。
彼は何をしていたのか思い出せなかった。
```

### 5. 物を主語にする

```
【従来】
彼はコーヒーを飲んだ。

【新感覚派的】
コーヒーが彼の喉を通過した。
カップは空になることを待っていた。
```

## 参考：技法の組み合わせ

複数の技法を組み合わせた例文：

```
正午。（体言止め＋現在形）
オフィスは昼食に飢えていた。（擬人化＋物質化）
或る者はコンビニへ走った。或る者はデスクで弁当を開けた。（並列スケッチ）
彼の空腹だけが、取り残されていた。（物質化＋主語の転倒）
```

## 芥川「歯車」との対比

| 歯車 | 頭ならびに腹 |
|------|--------------|
| 一人称の内面 | 三人称の外形 |
| 意味の過剰 | 感覚の羅列 |
| 偶然が必然に見える | 事物がただ並置される |
| 閉じた自意識 | 開かれた群衆 |
| 狂気へ向かう | 匿名性へ向かう |

両者を学ぶことで、「内面から書く」と「外形から書く」の両方の技法が身につく。
