## 情景描写

### 冒頭の技法

> 乙女たちの一団は水甕を頭に載せて、小丘の中腹にある泉の傍から、唄いながら合歓木の林の中に隠れて行った。

映画的なロングショット。人物の導入としての群衆描写。

### 夜の描写

> 夜は暗かった。卑弥呼は鹿の毛皮に身を包んで宮殿からぬけ出ると、高倉の藁戸に添って大兄を待った。栗鼠は頭の上で、栗の梢の枝を撓めて音を立てた。

短文。静寂の中の小さな音。

### 戦場の描写

> 忽ち、動揺く人波の点々が、倒れ、跳ね、躍り、渦巻くそれらの頭上で無数の白い閃光が明滅した。

動詞の連打。カオス的な戦場。

## 創作への応用

### 1. 古代風の文体を作る

**語尾の変換**
```
です/ます → である/なり
だった → であった
〜した → 〜せり
```

**人称の変換**
```
私 → 我
あなた → 爾
彼/彼女 → 彼の者/かの女
```

### 2. 暴力を美学化する

```
【普通】
彼は剣で敵の腕を切り落とした。

【日輪風】
敵の片腕は剣とともに宙を舞い、血の花を散らしながら草の上へ落ちた。
```

### 3. 短文で緊張感を出す

```
待て。
行け。
来たれ。
剣を抜け。
```

### 4. 自然描写を織り込む

```
【挿入前】
彼女は彼を待っていた。

【挿入後】
彼女は彼を待っていた。梟が木の上から降りて来た。野兎が茂みの中で跳ねた。月は昇った。
```

### 5. 色彩で象徴を作る

```
卑弥呼 → 紅（生命力、権力）
長羅 → 蒼白（死、執着）
月 → 銀（冷たさ、神秘）
血 → 朱（暴力、美）
```

## 参考：原文からの抽出

> 乙女たちの一団は水甕を頭に載せて、小丘の中腹にある泉の傍から、唄いながら合歓木の林の中に隠れて行った。

> 太陽は入江の水平線へ朱の一点となって没していった。

> 使部の片手は剣を握ったまま胴を放れて酒の中へ落ち込んだ。

> 「ああ、大神はわれの手に触れた。われは大空に昇るであろう。地上の王よ。我れを見よ。我は爾らの上に日輪の如く輝くであろう。」

> 彼女の頬に付いていた白い羽毛の一端が、反絵の呼吸のために揺れていた。

> 長羅の蒼ざめた額は地に垂れた。「卑弥呼、卑弥呼。」彼は恰も砂に呟くごとく彼女を呼ぶと、彼の瞼は閉じられた。
