# 日本文学ノート

日本近代文学の歴史を理解し、創作に活用するためのリポジトリ。

## モチベーション

- 本屋大賞など最近の小説に読み応えを感じないので、昔の小説をもっと読みたい
- 文学史の中でどう繋がっているか、時系列的な見取り図が欲しい
- 時代や派閥ごとに1冊ずつ読んで、自分の言葉で語れるようになりたい

## ディレクトリ構成

```
nihon-bungaku/
├── timeline.md          # 年表・時系列
├── eras/                # 時代別
│   ├── meiji.md         # 明治
│   ├── taisho.md        # 大正
│   └── showa-prewar.md  # 昭和戦前
├── movements/           # 文学運動・派閥別（時代順）
│   ├── 01-shajitsushugi.md  # 写実主義
│   ├── 02-genbun-icchi.md   # 言文一致運動
│   ├── 03-naturalism.md     # 自然主義
│   ├── 04-anti-naturalism.md # 反自然主義
│   ├── 05-shirakaba.md      # 白樺派
│   ├── 06-tanbi.md          # 耽美派
│   ├── 07-shinshicho.md     # 新思潮派
│   ├── 08-proletarian.md    # プロレタリア文学
│   ├── 09-modernism.md      # モダニズム文学
│   └── 10-burai.md          # 無頼派
├── authors/             # 作家別
└── works/               # 作品ノート
```
